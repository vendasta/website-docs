# WordPress Hosting Partner Guide

This guide covers everything partners need to know about selling and managing WordPress Hosting for their clients.

## How to Start Selling WordPress Hosting

1. Log into **Partner Center**.
2. Go to **Marketplace > Discover Products**.
3. Search for **WordPress Hosting**.
4. Click **Start Selling**.
5. Click **Done** to publish it in your Store.

## Access Control and White-Label Branding

### Manage Internal Access

1. Go to **Partner Center > Marketplace > Products > WordPress Hosting**
2. Click **Product Info**
3. Under **Access Control**, restrict access to team members only

Only Partner Center admin users with access to all markets can view the Admin Dashboard.

### White-Labeling the Product

1. In **Product Info**, locate **White-Label Branding**
2. Customize product name and upload your brand's logo

Note: White-label options are available on select subscription tiers.

## Partner-Specific Features

### Advanced Tools Visibility Control

Partners can control whether clients see advanced tools like phpMyAdmin, SFTP, and PHP logs. See [Feature Visibility & Access Control](./feature-visibility-control.md) for details.

### Template Visibility Control

Partners can hide pre-built templates from clients. See [Template Visibility Control](../vendasta-specific/template-visibility-control.md) for configuration steps.

## Client Onboarding Process

### Step 1: Account Creation

Create accounts for each client through Partner Center:
- Go to **Accounts > Manage Accounts**
- Add client business information
- System will auto-populate additional details

### Step 2: Product Activation

1. Select the client account in Partner Center
2. Click **Order Products** and search "WordPress Hosting"
3. Choose between Standard and Pro based on client needs
4. Complete activation process

### Step 3: Site Setup

Once activated, help clients with:
- Choosing subdomain or connecting custom domain (Pro only)
- Selecting appropriate template
- Basic site configuration

## Billing and Plan Management

### Plan Upgrades/Downgrades

- Upgrades are immediate
- Downgrades may have restrictions based on current usage
- Changes are processed through Partner Center

### Resource Monitoring

Monitor client usage for:
- Storage limits
- Bandwidth usage
- Performance metrics
- Feature utilization

## Support and Troubleshooting

### Partner Dashboard Access

Partners have enhanced access to:
- Client site analytics
- Performance metrics
- Error logs and diagnostics
- Backup management

### Common Client Issues

**Domain Connection Problems:**
- Verify DNS settings
- Check SSL certificate status
- Confirm domain ownership

**Performance Issues:**
- Review PageSpeed metrics
- Check for plugin conflicts
- Optimize images and caching

**Email Delivery Problems:**
- Review Email History logs
- Check SPF/DKIM records
- Verify SMTP configuration

## Best Practices

### Client Communication

- Set clear expectations about features available in each plan
- Provide training on WordPress basics
- Establish support channels and escalation procedures

### Site Management

- Regular backup schedules
- Keep WordPress core, themes, and plugins updated
- Monitor security and performance metrics
- Use staging environments for testing changes

### Scaling Considerations

- Monitor resource usage trends
- Plan for seasonal traffic spikes
- Consider Pro upgrades for growing businesses
- Implement caching strategies for high-traffic sites

## Frequently Asked Questions

<details>
<summary>Can I create sites for clients directly?</summary>

Yes, partners can create and manage sites on behalf of clients through Partner Center. Clients can then access their sites through the Business App.
</details>

<details>
<summary>How do I handle client requests for features not included in their plan?</summary>

Review the feature comparison and recommend appropriate plan upgrades. Many advanced features are available in WordPress Hosting Pro.
</details>

<details>
<summary>What happens if a client exceeds their resource limits?</summary>

The system will notify both partner and client. You can help them optimize their site or recommend upgrading to a higher plan.
</details>

<details>
<summary>Can I customize the WordPress Hosting interface for my brand?</summary>

Yes, white-label options are available on select subscription tiers. You can customize the product name and add your brand logo.
</details>
